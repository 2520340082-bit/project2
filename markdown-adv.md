### 2\. 표(테이블)

마크다운으로 표를 만들 수 있어 데이터를 깔끔하게 정리하기 좋습니다.

| 헤더 1 | 헤더 2 | 헤더 3 |
|:---:|:---|---:|
| 가운데 정렬 | 왼쪽 정렬 | 오른쪽 정렬 |
| 데이터 A | 데이터 B | 데이터 C |
| 예시 1 | 예시 2 | 예시 3 |

**정렬 방식**:

  * `:---:` : 가운데 정렬
  * `:---` : 왼쪽 정렬
  * `---:` : 오른쪽 정렬
